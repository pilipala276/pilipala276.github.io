触摸电路

电容触摸按键芯片的工作原理为：对于电容触摸按键（其原理如下图3所示）来说，当人体（手指）触摸按键的表面时，由于人体相当于一个接大地的电容，因此会加大按键感应片的对地电容，电容触摸按键芯片即通过检测感应片对地电容的变化，判断出是否有触摸的动作。

查阅TTP223-BA6的数据手册可得其各管脚定义如下表，可知其使用方式I连接到铜皮，触摸时形成对地电容并被TTP223-BA6检测到，Q按照TOG和AHLB的配置内容输出

Pad Name Pad Description
Q CMOS output pin
VSS Negative power supply, ground
I Input sensor port
AHLB utput active high or low selection, 1=>Active low; 0(Default)=>Active high
VDD Positive power supply
TOG Output type option pin, 1=>Toggle mode; 0(Default)=>Direct mode

FLASH

Flash 存储器（FLASH EEPROM）又称闪存，快闪。它是EEPROM的一种。它结合了ROM和RAM的长处。不仅具备电子可擦除可编辑（EEPROM）的性能，还不会断电丢失数据同时可以快速读取数据。它于EEPROM的最大区别是，FLASH按扇区（block）操作，而EEPROM按照字节操作。FLASH的电路结构较简单，同样容量占芯片面积较小，成本自然比EEPROM低，因此适合用于做程序存储器。
ESP32支持SPI对外部 flash 数据读取、写入、擦除和内存映射操作

电源管理（充电电路，切换供电，降压电路）

锂电池在充电时，充电管理电路是一个极其复杂的过程，既要保证锂电池能够充满，又要保证锂电池的性能，最重要的是要保证锂电池不能过充。
整个充电电路应该具有以下几种充电模式：低电压预充电模式;全速充电模式;涓流充电模式;顶端截止、脉冲充电模式;充电截止模式。
因此通常用电磁电源管理芯片来进行充电操作。
这里采用的是TP4054

采用线性稳压芯片ME6210A33PG降压到3.3V

充电电路及供电切换电路如下在有外部供电即+5V时，P沟道MOS管Q1关闭，由+5V供电，同时+5V通过TP4054即U12对电池充电
